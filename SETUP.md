# Setup - Was du brauchst

Bevor du das Projekt nutzen kannst, installiere bitte diese Dinge. Keine Sorge, ist ganz einfach!

## 1. VS Code

Das ist der Editor, mit dem du das Notebook öffnest.

- Download: https://code.visualstudio.com/
- Einfach installieren wie jedes andere Programm

## 2. Python

Das ist die Programmiersprache, die das Projekt nutzt.

- Download: https://www.python.org/downloads/
- **Wichtig:** Beim Installieren "Add Python to PATH" ankreuzen!
- Nach der Installation überprüfen: Terminal öffnen und `python --version` eingeben

## 3. Python Packages installieren

Das sind die Zusatz-Bibliotheken, die wir brauchen.

Terminal öffnen und folgendes eingeben:
```bash
pip install pandas openpyxl selenium webdriver-manager
```

Das war's! Python lädt sich die Pakete jetzt automatisch runter.

## 4. Chrome Browser

Das Programm nutzt Chrome zum Scrapping.

- Download: https://www.google.com/chrome/
- Normale Installation, kein Setup-Zeugs nötig

## Fertig!

Du kannst jetzt das Projekt nutzen. Siehe dazu die Datei `ANLEITUNG.md`.
